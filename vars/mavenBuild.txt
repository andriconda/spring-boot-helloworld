Maven Build Pipeline

This shared library provides a reusable Maven build pipeline.

Usage:
------
@Library('spring-boot-shared-lib') _

mavenBuild()

Or with custom configuration:

@Library('spring-boot-shared-lib') _

mavenBuild(
    gitUrl: 'https://github.com/your-org/your-repo.git',
    gitBranch: 'develop',
    mavenGoals: 'clean install',
    skipTests: false,
    mavenTool: 'Maven'
)

Parameters:
-----------
- gitUrl: Git repository URL (default: 'https://github.com/andriconda/spring-boot-helloworld.git')
- gitBranch: Git branch to checkout (default: 'main')
- mavenGoals: Maven goals to execute (default: 'clean package')
- skipTests: Skip tests during build (default: true)
- mavenTool: Name of Maven tool in Jenkins (default: 'Maven')
